## Step 5: Review Architecture Decisions

Browse the [ADR directory](../architecture/decisions/). Do not attempt to change an ADR
decision without opening a new ADR first.

## Key Principle

This project is documentation-first. If something is not documented, it does not officially
exist. If you make a decision that is not recorded, you are creating undocumented institutional debt.

Document first, code second.
