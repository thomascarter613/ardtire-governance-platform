# Code of Conduct

## Our Commitment

The Ardtire Governance Platform is a project serving a constitutional institution.
Participation in this repository — through issues, pull requests, or discussion —
is governed by the same standards of conduct expected within the Ardtire Society.

## Expected Behaviour

- Communicate with precision and respect.
- Engage with technical disagreements through evidence and reasoning.
- Accept that architectural and governance decisions have an established process.
- Protect the confidentiality of sensitive governance matters.

## Unacceptable Behaviour

- Harassment, personal attacks, or discriminatory language.
- Publishing private information about others without consent.
- Submitting deliberately misleading information in issues or PRs.
- Attempting to subvert the governed decision-making process through repository activity.

## Enforcement

This repository is maintained by a single engineer and institution founder.
Violations will result in removal of contributions and restriction of access.

Contact: thomas.carter@appliedinnovationcorp.com

## Scope

This code of conduct applies to all interactions within this repository —
issues, pull requests, discussions, and code review.
